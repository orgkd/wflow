addErrorStage(options)

Adds fail stage to the workflow. With adding fail stage also will be created links from intermediate stages of workflow. To prevent creation of such links use failable=false in stage options. It's assumed that workflow cannot fail on START and SUCCESS stages. You can customize this behavior by creating stages using Chain constructor of this.addStage() method.
Parameters:
Name Type Description
options object Error stage options.
